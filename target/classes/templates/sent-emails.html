How this page works

There is no background scheduler in the app yet: notifications are sent when an admin clicks Send on a row below (or sends a test from Notifications).

The grid lists every planned row for the selected goals year (six periods, manager row plus three employee reminders each). Rows sent in January use annual goals-setup templates; later dates in the first period use normal check-in templates (configure everything under Notifications). Use the employee filter to narrow the table; the raw log filters by planned key. Repeat sends overwrite the latest status for the same key.
